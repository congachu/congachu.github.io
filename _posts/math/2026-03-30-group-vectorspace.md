# 군

## 군 (Group)

- **군의 정의**  
  어떤 집합 G와 이항 연산 ⊗ : G × G → G 가 다음 네 가지 성질을 만족하면 (G, ⊗)를 군이라고 한다.

- **조건**
  - 닫힘성 (Closure)  
    모든 x, y ∈ G에 대해 x ⊗ y ∈ G
  - 결합법칙 (Associativity)  
    (x ⊗ y) ⊗ z = x ⊗ (y ⊗ z)
  - 항등원 (Identity)  
    x ⊗ e = e ⊗ x = x
  - 역원 (Inverse)  
    x ⊗ y = y ⊗ x = e 를 만족하는 y 존재

- **군의 의미**  
  연산을 계속 수행해도 구조가 유지되고, 되돌릴 수 있으며, 기준이 되는 항등원이 존재하는 시스템

- **예시**
  - 회전
  - 평행이동
  - 순열

---

## 아벨리안 군 (Abelian Group)

- **정의**
  ```
  x ⊗ y = y ⊗ x
  ```
  교환법칙까지 만족하는 군

- **예시**
  - (Z, +) → 아벨리안 군
  - (N₀, +) → 군 아님 (역원 없음)
  - (Z, ·) → 군 아님
  - (R*, ·) → 아벨리안 군
  - 가역행렬 집합 → General Linear Group

# 벡터 공간

## 벡터 공간 (Vector Space)

- **정의**
  ```
  V = (V, +, ·)
  ```

- 연산
  - 벡터 덧셈: V × V → V
  - 스칼라 곱: R × V → V

---

## 벡터 공간 조건

1. (V, +)는 아벨리안 군
2. λ(x + y) = λx + λy
3. (λ + μ)x = λx + μx
4. λ(μx) = (λμ)x
5. 1x = x

---

## 벡터 공간 예시

- Rⁿ
- R^(n×m)
- 복소수 공간

---

# 벡터 부분 공간

## 정의

V가 벡터 공간이고 U ⊆ V 일 때  
U가 벡터 공간 조건을 만족하면 부분공간

---

## 체크 방법

다음 3개만 확인

- 0 ∈ U
- x + y ∈ U
- λx ∈ U

---

## 문제

- Ax = 0 → 부분공간 (O)
- Ax = b → 부분공간 (X)

---

# 선형 조합

## 정의

```
v = λ₁x₁ + ... + λₖxₖ
```

---

## 예시

```
(1,0,0)
(0,1,0)
(0,0,1)

→ (5,3,2) 가능
```

---

# 선형 독립 / 종속

## 정의

```
0 = λ₁x₁ + ... + λₖxₖ
```

---

## 선형 독립

```
λ₁ = λ₂ = ... = 0
```

유일한 해만 존재

---

## 선형 종속

- 0이 아닌 계수 존재

- 하나의 벡터가 다른 벡터로 표현 가능

---

## 성질

- 영벡터 포함 → 종속
- 벡터 하나라도 표현 가능 → 종속
- n차원에 n개 초과 → 종속

---

## 예시

### 선형 독립

```
(1,0,0)
(0,1,0)
(0,0,1)
```

---

### 선형 종속

```
(1,0)
(5,0)
```
