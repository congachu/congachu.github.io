# 벡터

## 벡터의 정의와 성질

벡터 공간 $V$ 에는 덧셈 연산 $+$ 와 스칼라 곱이 정의된다.

### 덧셈

- 항등원: $0 \in V$ 가 존재하여  
  $v + 0 = v$
- 역원: $-v \in V$ 가 존재하여  
  $v + (-v) = 0$
- 교환법칙: $u + v = v + u$
- 결합법칙: $(u + v) + w = u + (v + w)$

### 스칼라곱

- 분배법칙  
  $a(u + v) = au + av$  
  $(a + b)v = av + bv$
- 결합법칙  
  $a(bv) = (ab)v$
- 항등원  
  $1v = v$

---

## 벡터의 종류

- 기하학적 벡터
- 다항식
- 오디오 신호
- $\mathbb{R}^n$ 공간의 점

# 선형대수

선형대수는 벡터의 덧셈과 스칼라 곱을 다루는 학문이다.

---

## 선형연립방정식

```
Ax = b
```

- A: 계수 행렬  
- x: 미지수 벡터  
- b: 결과 벡터  

---

## 해의 종류

| 유형 | 의미 |
|---|---|
| 해 없음 | 평행 |
| 유일한 해 | 한 점에서 교차 |
| 무한 해 | 동일한 직선 |

---

## 행렬

행렬은 숫자를 직사각형 형태로 배열한 것이다.

- (1, n): 행 벡터
- (m, 1): 열 벡터

---

## 행렬 연산

### 덧셈
같은 위치의 값끼리 더한다.

### 곱셈

```
(AB)[i][j] = Σ A[i][k] * B[k][j]
```

주의:  
```
AB != BA
```

---

## 항등 행렬

```
I = 단위행렬
A * I = A
```

---

## 역행렬

```
A^-1
```

조건

```
A * A^-1 = I
```

2x2 역행렬

```
| a b |
| c d |

→

1 / (ad - bc) *

| d  -b |
| -c  a |
```

단,

```
ad - bc != 0
```

---

## 전치행렬

```
A^T
```

행과 열을 뒤집은 것

---

## 주요 성질

```
(AB)^-1 = B^-1 A^-1
(AB)^T = B^T A^T
(A^T)^T = A
(A + B)^T = A^T + B^T
```

---

## 선형연립방정식 풀이

```
x = xp + xh
```

- xp: 특수해
- xh: 동차해

```
x = (A^T A)^-1 A^T b
```

- 계산량 큼
